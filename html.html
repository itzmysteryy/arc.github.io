Introduction to HTML


What is HTML

HTML is the language used to structure content on the web. It helps you style text, insert images, make links, create forms, and organize information. Everything is saved in a text file that any web browser can open. HTML relies on tags to define the content's structure.


Features of HTML

  • It is platform-independent. For example, Chrome displays the same pages identically across different operating systems such as Mac, Linux, and Windows.

  • Images, videos, and audio can be added to a web page (For example - YouTube shows videos on their website)

  • HTML is a markup language and not a programming language.

  • It can be integrated with other languages like CSS, JavaScript, etc. to show interactive (or dynamic) web pages
A beautiful analogy to understand HTML, CSS, and JavaScript:

Creating a webpage is like building a car. HTML is the framework, like the car's skeleton, setting the structure. CSS is the paint and style, making it look good with colors and design. JavaScript acts as the engine, providing functionality and making things work, just like a car's engine powers it. In simple terms, HTML shapes, CSS beautifies, and JavaScript adds life and action, much like assembling different car parts to create a working vehicle.

Create your first website

Lets continue the course by making a website about yourself!


HTML Document Structure

Every HTML document follows a specific structure. Here's an example of a simple HTML document structure:

                
                  <!DOCTYPE html> 
                  <html> 
                  <head> 
                  <title>Your Page Title</title> 
                  </head>
                  <body> 
                  <h1>Welcome to My Page 
                  <p>This is some introductory text about me.
 
                  </body>
                  </html>

Your First HTML Page

Now, let's create your first HTML page:

  1. Open a new file in your text editor (e.g., VS Code) and name it index.html.
  2. Inside your index.html file, use the HTML structure as shown above.
  3. Edit the title tag to give your page a unique title.
  4. Within the <h1> tags, add a suitable heading.
  5. Below the heading, use the <p> tag to write a brief introduction about yourself. You can include details about your interests, skills, or anything you'd like to share.

Adding Style

You can make your page look more appealing by adding some CSS (Cascading Style Sheets). Here's an example of how to link an external CSS file to your HTML:

            <code>
            <link rel="stylesheet" type="text/css" href="styles.css">
            </code>

Create a styles.css file to apply styles to your page. You can change the background color, text styles, and layout to make your page unique.

Viewing Your Page

After creating your HTML and CSS files, you can view your web page by using a local development server or simply opening the index.html file in your web browser.
